# Operating System Simulation (OS-Sim)
This project is a comprehensive simulation of core Operating System concepts, developed as a major term project for the Operating Systems (CS302) course at Ankara Bilim University. It focuses on how an OS manages processes, schedules tasks, and handles memory allocation.

# Key Modules
Process Management: Simulation of process states (Ready, Running, Waiting) and PCB (Process Control Block) structures.

CPU Scheduling: Implementation of scheduling algorithms to manage CPU time efficiently.

Memory Management: Simulation of memory allocation strategies and tracking memory usage to prevent fragmentation.

Synchronization: Handling inter-process communication and preventing race conditions using C++ classes.

# Tech Stack
Language: C++

Paradigm: Object-Oriented Programming (OOP)

Concepts: Multi-threading, Data Structures (Queues, Linked Lists), Logic Design.

# Why This Project?
This was developed as a course assignment to gain hands-on experience with low-level system operations.
